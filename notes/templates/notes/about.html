Описание проекта

Данный сайт представляет собой полноценное веб-приложение, разработанное на Django с использованием современных DevOps-практик. Проект реализ как учебный, но с промышленным уровнем deployment.

 

Технологический стек:

  • Backend: Django (Python) с системой аутентификации, ORM и административной панелью

  • База данных: PostgreSQL с регулярными бэкапами

  • Инфраструктура:

    • Docker-контейнеризация (Docker Compose)

    • Nginx как reverse proxy с HTTPS

    • Fail2ban для защиты от bruteforce-атак

  • Дополнительные функции:

    • Система комментариев с модерацией (пока в режиме тестирования)

    • Отправка email-уведомлений (регистрация, обратная связь)

Особенности реализации:

  • Автоматическое развертывание через CI/CD (GitHub Actions) (пока в режиме тестирования)

  • Контейнеры настроены на автозапуск при перезагрузке сервера

  • Оптимизированная работа со статикой (Nginx + Django)

  • Защита от DDoS через Nginx rate limiting

Планы по развитию:

  • Добавление API на DRF (Django REST Framework)

  • Интеграция Celery для фоновых задач

  • Реализация полнотекстового поиска

  • Настройка мониторинга (Prometheus + Grafana)
